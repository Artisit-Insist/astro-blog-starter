이 폰트는 OG(소셜 공유) 이미지의 한글 제목 렌더링에만 쓰입니다.
다른 폰트로 바꾸려면 이 폴더의 `.otf`(또는 `.ttf`)를 교체하고
`src/pages/og/[...slug].png.ts`의 폰트 경로·이름을 수정하세요.

이 저장소 본문은 MIT 라이선스이지만, 폰트 파일은 위 OFL을 따릅니다.
